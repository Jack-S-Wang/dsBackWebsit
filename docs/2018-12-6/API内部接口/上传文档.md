###上传文档

**简要描述：** 

-保存文档并上传文档路径。

**请求URL：** 

-`http://127.0.0.1:13100/uploadDoucment`

**请求方式：**

- POST 

**请求头:**

-"content-type":"multipart/form-data"

**参数：** 

| 参数名  | 必选 | 类型     |说明|
| ------ | -------- | -------- |------|
|file |	是 |	File| 	需要上传的文件|

**请求事例**

 **返回示例**
 
```
  成功
          {
      "data": "127.0.0.1:13100/doucmentFiles/..",
       "code": 101000
    }
```

``` 
失败
        {
      "data": "执行失败",
       "code": 101004
    }
```

**返回参数说明**

| 参数名  |   类型     |说明|
| ------ | -------- |------|
| code | int |返回码|
|data|String|返回的数据|
