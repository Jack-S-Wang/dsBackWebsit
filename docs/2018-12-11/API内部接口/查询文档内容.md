###查询文档内容

**简要描述：** 

-查找文档为ID值的文档内容。

**请求URL：** 

-`http://127.0.0.1:21010/content`

**请求方式：**

- POST 

**请求头:**

-"content-type":"application/json"

**参数：** 

| 参数名  | 必选 | 类型     |说明|
| ------ | -------- | -------- |------|
|garde |	是 |	String |	等级目录标识，指文档或其他目录所在的上级目录等级，1或2|
|id |	是 |	String |	文档的ID值|
|s_id |	否 |	String |	二级目录的ID值，当garde为1时不需要，为2时必填|

**请求事例**

```
          {
      "garde": "1",
       "id":"6746862h68768dg638"
    }
```

 **返回示例**
 
```
  成功
          {
      "data": "文本内容",
       "code": 101000
    }
```

```   
失败
          {
      "data": "执行失败",
       "code": 101004
    }
```

**返回参数说明**

| 参数名  |   类型     |说明|
| ------ | -------- |------|
| code | int |返回码|
|data|String|返回的数据|
