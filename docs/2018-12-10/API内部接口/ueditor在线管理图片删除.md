###ueditor在线管理图片删除

**简要描述：** 

-ueditor在线管理图片删除指定图片

**请求URL：** 

-`http://127.0.0.1:11010/delImagesByPath`

**请求方式：**

- POST 

**请求头:**

-"content-type":"application/json"

**参数：** 

| 参数名  | 必选 | 类型     |说明|
| ------ | -------- | -------- |------|
|path| 	是 |	String |	图片相对于项目位置的相对路径|

**请求事例**

```
          {
      "path": "images/upload/image/xxx.png"  
    }
```

 **返回示例**
 
``` 
  成功
          {
      "data": "执行成功",
       "code": 101000
    }
```

```   
失败
          {
      "data": "执行失败",
       "code": 101004
    }
```

**返回参数说明**

| 参数名  |   类型     |说明|
| ------ | -------- |------|
| code | int |返回码|
|data|String|返回的数据|
