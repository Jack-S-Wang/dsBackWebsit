###上传轮播图

**简要描述：** 

-官网轮播图上传接口

**请求URL：** 

-`http://127.0.0.1:11010/uploadFile`

**请求方式：**

- POST 

**请求头:**

-"application/x-www-form-urlencoded;charset=utf-8"

**参数：** 

| 参数名  | 必选 | 类型     |说明|
| ------ | -------- | -------- |------|
|id|否| 	string| 	图片id，如果要更换指定图片需要id|
|file|是| 	File| 	图片资源 |

**请求事例**

 **返回示例**
 
```
  成功
          {
      "data": "192.168.11.197:13100/doucmentFiles/1543457013402.pdf",
       "code": 101000
    }
```

``` 
失败
          {
      "data": "执行失败",
       "code": 101004
    }
```

**返回参数说明**

| 参数名  |   类型     |说明|
| ------ | -------- |------|
| code | int |返回码|
|data|String|返回的数据|
