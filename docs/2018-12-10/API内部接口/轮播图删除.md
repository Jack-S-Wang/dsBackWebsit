###轮播图删除

**简要描述：** 

-通过id删除目标轮播图

**请求URL：** 

-`http://127.0.0.1:11010/deleteImag`

**请求方式：**

- GET 

**请求头:**

-"content-type":"application/json"

**参数：** 

**请求事例**

```
     http://127.0.0.1:13100/deleteImag?id=xxxx
```

 **返回示例**
 
```
  成功
          {
      "data": "删除成功",
       "code": 101000
    }
```

```   
失败
          {
      "data": "执行失败",
       "code": 101004
    }
```

**返回参数说明**

| 参数名  |   类型     |说明|
| ------ | -------- |------|
| code | int |返回码|
|data|String|返回的数据|
