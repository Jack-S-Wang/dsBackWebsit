###删除文档

**简要描述：** 

-根据对应id删除文档。

**请求URL：** 

-`http://127.0.0.1:21010/delDocument`

**请求方式：**

- POST 

**请求头:**

-"content-type":"application/json"

**参数：** 

| 参数名  | 必选 | 类型     |说明|
| ------ | -------- | -------- |------|
|garde |	是 |	String |	等级目录标识，指文档或其他目录所在的上级目录等级，1或2|
|s_id |	否 |	String |	当garde为2时，需要传入二级目录ID|
|id |	是 |	String |	文档id|

**请求事例**

```
            {
      "garde": "1",
       "id": "673t4288s83h67"
    }
```

 **返回示例**
 
```
  成功
       {
      "data": "执行成功",
       "code": 101000
    }
```

``` 
失败
        {
      "data": "执行失败",
       "code": 101004
    }
```

**返回参数说明**

| 参数名  |   类型     |说明|
| ------ | -------- |------|
| code | int |返回码|
|data|String|返回的数据|
