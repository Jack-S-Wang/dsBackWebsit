###删除上传的文档信息

**简要描述：** 

-删除上传失误的文档

**请求URL：** 

-`http://127.0.0.1:21010/delDoucmentLoad`

**请求方式：**

- POST

**请求头:**


**参数：** 

| 参数名  | 必选 | 类型     |说明|
| ------ | -------- | -------- |------|
|lod_id |	是 |	String| 	删除对应上传文档id|
|id|是|String|所对应的文档id|
|garde|是|String|文档所对应的目录级别|
|s_id|否|String|当garde为2时必须传入的二级目录级别|

**请求事例**

```
{
	"lod_id":"1544775055828",
	"id":"5c1203deadc16416f875f455",
	"garde":"0"
	}
```

**返回示例**
 
```
  成功
        {
    "code": 101000,
    "data": "执行成功"
}
```

```
失败
           {
      "data": "执行失败",
       "code": 101004
    }
```

**返回参数说明**

| 参数名  |   类型     |说明|
| ------ | -------- |------|
| code | int |返回码|
|data|String|返回的数据|
